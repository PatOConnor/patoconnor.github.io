DovahPy

This program is a set of tools intended to make playing skyrim on PC easier

The idea is that instead of looking up a bunch of individual items when making console commands, you can use command builder to generate them all at once.

The alchemist package briefly inputs keypresses to get your inventory info, so it needs to be executed with administrator privileges.

Features

  • Skyrim Scraper
    • Scrapes uesp and skyrimcommands to download a dataset of skyrim data
  • Lookup
    • Search through a data set of the various things you can put into a skyrim console command
  • Command Maker
    • Compile any console command from Skyrim
    • Create multiple commands in a row and save to one file
    • Can give nicknames to commands for reference
    • saves batch file of made commands to skyrim directory where it can immediately be run in-game
  • Alchemist
    • Gets a list of items in your skyrim inventory by taking screenshots and parsing image-to-text with the pyTesseract library
    • Passes that list into an algorithm that prints out the potions you should craft for maximum revenue

Technologies

  • BeautifulSoup
  • Selenium Webdriver
  • textualize/rich (Terminal GUI)
  • PyAutoGUI
  • OpenCV2
  • Tesseract
  • regex
