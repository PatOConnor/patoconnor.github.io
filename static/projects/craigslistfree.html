Cambridge Free Stuff Finder

This is a script I wrote for a friend that checks your local craiglist to see if there's any new free stuff.

Features

  • performs an automated search of craigslist
  • able to do multiple search queries and display all results

How it Works

  • it keeps a small local database of items it has already scanned for, so it doesn't display any duplicates.
  • You set your postal code, distance radius, subdomain, and any search phrases in the config file
  • In the config file, set RESET equal to True to delete pre-existing data from the database.
  • Note: Has only been tested for USA craigslist

Technologies

  • BeautifulSoup
  • textualize/rich
