Rockler Accessor

This script automates the process of looking up the inventory at an individual location of Rockler Lumber and Woodworking.

Features

  • filter through a large data set
  • Acquires dataset from the web

How it Works

The rockler accessor uses selenium to remotely click the "check local inventory" button on the rockler website to display the table of info. It proceeds to use BeautifulSoup to parse the table into a data structure, and the user can input parameters to filter to the wood they'd prefer.

There is a Command-Line version of the program, and a GUI version as well

Technologies

  • BeautifulSoup
  • Selenium Webdriver
  • textualize/rich (Terminal GUI)
  • PyQt (GUI)
